
## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/yavuzhanaymak/Kisisel-is-takip.git
```

Proje dizinine gidin

```bash
  cd Kisisel-is-takip
```

Gerekli paketleri yükleyin

```bash
 yarn install
```

Sunucuyu çalıştırın

```bash
  npm run build && npm start
```
